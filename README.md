# 🚀 Intervuary.AI  
### AI-Powered Resume-Driven Mock Interview Platform

> Upload your resume → Practice adaptive mock interviews → Get intelligent feedback → Track measurable improvement.

Intervuary.AI is a production-ready SaaS platform that transforms traditional interview preparation into a structured, AI-driven experience. Instead of generic questions, it generates **role-specific interview simulations** directly from your resume and delivers real-time evaluation with actionable insights.

---

## 🌟 Why Intervuary.AI?

Most candidates practice randomly.  
Intervuary.AI makes practice:

- 🎯 Targeted to your resume & experience  
- 🧠 AI-generated and adaptive  
- 📊 Measurable with detailed feedback reports  
- 💬 Realistic with timed voice-based simulations  

It creates a repeatable improvement loop that builds confidence and technical depth.

---

## ✨ Features

- 📄 **Resume Upload (PDF)**  
  Extracts skills and projects to generate tailored interview questions.

- 🎙 **Smart Voice Mock Interviews**  
  Record answers, simulate real pressure, and reattempt responses.

- 🧩 **Multiple Interview Modes**
  - HR Interview Mode
  - Technical Mode
  - Confidence & Tone Analysis
  - Timed Simulation Mode

- 📊 **AI Answer Evaluation**
  - Communication scoring
  - Technical correctness assessment
  - Confidence analysis

- 📑 **Downloadable PDF Reports**
  - Strengths & weaknesses
  - Improvement suggestions
  - Performance summary

- 💳 **Credit-Based Access System**
  - Purchase credits via Razorpay
  - Secure checkout flow
  - Credit history tracking

- 🔐 **Secure Authentication**
  - Firebase Google Authentication
  - Protected routes
  - User session history

- 🎨 **Modern SaaS UI**
  - Built with React & Tailwind
  - Framer Motion animations
  - Responsive and clean design
